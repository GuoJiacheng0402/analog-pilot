## 约定速查表

### 连接 / 环境
- `.env` 中 **`APILOT_DISABLE_CONTROL_MASTER=true` 必须开启**（否则远端 Python 探测返回空字符串）。
- `APILOT_REMOTE_SCRATCH_ROOT` 应指向**家目录**下的固定目录（避免使用 `/tmp`，重启后会被清空）。
- `APILOT_CADENCE_CSHRC` 在 SKILL 模式下**留空**；仅在使用 Spectre 独立通道时填写，且必须为 **csh 语法**。
- 直接运行 `virtuoso` 报 `libap_sh.so` → 需先 `source bashrc_cds` 再 `source bashrc_mentor`。
- `bashrc_cds` 设置的 `LD_PRELOAD` 会影响当前 shell → 用于启动 Virtuoso 的 shell 不应执行其他操作，且不应写入 `~/.bashrc`。
- 在已验证的课程服务器环境中，SSH 端口 22 可达；桥接的隧道端口由用户名哈希自动分配，通常无需手动指定。

### SKILL / GUI
- **任何弹出模态表单的操作都会阻塞 daemon**。出现 `NO RESPONSE` 时，应首先检查 VNC 中是否有未关闭的对话框。
- 生成 symbol 应使用 `schPinListToSymbol(...)`，而非会弹出表单的 `schViewToView` 八参形式。
- **修改 schematic 前应确认没有处于活动状态的 maestro session**（否则会弹出 re-netlist 对话框导致阻塞）。
- 循环变量**不应命名为 `t`**（`t` 为受保护变量，报错会表现为括号不匹配，具有误导性）。
- 赋值号两侧需有空格：应写 `cc = ddGetObj(...)`，而非 `cc=ddGetObj(...)`。
- 本 PDK 中 schematic 的 viewType 为 `"schematic"`，而非 `"schematicSchematic"`。

### CSMC05_M3 PDK
- MOS 尺寸：**必须设置 `fw`（finger width），不能仅设置 `w`**；`simW = fw × fingers`。
- 电阻定值：使用 `calculatedParam="Length"` + `segL/segW`（浮点 SI 米），**而非** `r="5K"`。
- 物理参数需量化到 **0.01µm 网格**，否则 DRC 报 `offgrid_check`。
- vsource pulse 的高低电平字段为 **`val0`/`val1`**，而非 `v1`/`v2`（误用会被静默忽略）。
- pulse 参数名为 `td / tr / tf / twidth / per`（而非 delay/rise/fall/width/period）。
- 仿真模型：include `s05mixdtssa01v12.scs` 的 `section=tt`（MOS）与 `section=biptypical`（BJT）；
  **不应**再 include `section=bip`（会报 `Model already defined`）。
- BJT：调用 `qvp5`；面积比 `n` 通过**并联 n 个单位管**实现（`m` 仅服务于仿真，不放大版图几何）；
  LVS 中 BJT 的 subtype 为 **`P1`**（`area=25p`），而非 `qvp5`。
- 相位应使用 `phaseDegUnwrapped(...)`，不应自行计算 `(180/π)*phase(...)`（会得到数千度的错误值）。

### 后仿
- 前仿与报告截图应优先使用 **Spectre**（而非 HSPICE）；ADE 状态名包含 "spectre" 并不代表当前 simulator 为 Spectre，需在 Setup 中确认。
- 采用**两步 PEX**：先 **no-R/C**（验证流程与 PDK 文件配置正确性，应与前仿一致），再 **full R+C**（评估寄生影响）。
- source CDL 使用 **deck 的 subtype 名**（MOS：NP/NN；电阻：RL/H1；电容：CP；PNP：P1），而非仿真模型名。
- DRC 通过不代表无电气短路：**同层长 bus 横穿会导致 LVS 短路但 DRC 不报**；高阻输入网走 A2，电源/地/参考网保留在 T3。
- PEX 子电路的端口顺序与 schematic **不同**，testbench 需按各自顺序实例化。
- PEX 网表中 `rpoly2/rhr1k/cpip` 为提取器件名，Spectre 未定义 → 需在仿真副本中映射为内建 `resistor/capacitor`（**不修改原网表**）。
- BGR 测量 TC：使用**启动瞬态**逐点上电，取稳态末段平均（纯 DC 温扫会落入零电流分支，导致结果无效）。

### 协作
- 进入新 PDK 或新任务前，应先 **dump 真值**（CDF、几何、合法参数），不应凭记忆假设。
- **小步可回退**：每个里程碑做本地与远端双备份，并将状态写回日志。
- 在文档/日志顶部维护一个「当前状态」块，每次接手前先阅读。

## 命令速查（CLI）

```bash
apilot init <user@host>       # 生成 ~/.apilot/.env 模板
apilot start [-p PROFILE]     # 建立 SSH 隧道并部署 daemon（打印 CIW load 行）
apilot status [-p PROFILE]    # 查询 [tunnel] 与 [daemon]
apilot selftest [-p PROFILE]  # 一次性自检：ssh / 隧道 / daemon / SKILL / spectre
apilot stop [-p PROFILE]      # 关闭隧道
apilot restart [-p PROFILE]   # 重启
```

> 其余操作（截图、列窗口等）通过 `client.execute(...)` 下发对应 SKILL，或使用
> [`tools/grab_vnc_screenshot.sh`](../tools/grab_vnc_screenshot.sh) 截取 VNC 桌面。
